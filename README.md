# catalogo_peliculas

La aplicación cuenta con una página de inicio donde permite registrar usuarios, e iniciar sesión. Una vez iniciada sesión te permite visualizar el catálogo de películas, y al dar click en alguna de las mismas se visualiza una descripción completa de la película.
La aplicación cuenta también con un botón para agregar más películas y eliminar alguna ya existente.
Para iniciar la aplicación, clone el repositorio y desde el main.dart inicie el proyecto Flutter con Chrome o Edge. Para agregar películas asegurarse que las imagenes son .jgp, no admite .hmtl ni .webp.
